# Install to XAMPP

## Windows

```
git clone https://github.com/anpig/db-project.git C:/xampp/htdocs/db-project
```

## macOS

```
git clone https://github.com/anpig/db-project.git /Applications/XAMPP/xamppfiles/htdocs/db-project
```
